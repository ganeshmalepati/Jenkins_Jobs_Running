This repository contains Groovy script files that are used to configure and trigger Jenkins jobs. These scripts are hosted on GitHub and can be executed in Jenkins by providing the GitHub repository URL.

How It Works
- The Groovy scripts are stored in this GitHub repository.
- In Jenkins, you can trigger jobs by referencing the GitHub URL of this repository.
- Jenkins will access the scripts directly from the GitHub URL and execute them.
- This allows for centralized management and version control of Jenkins job configurations.
